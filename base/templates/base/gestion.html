Interface d'administration

Accès à l'interface d'administration
Cette interface permet de bidouiller tout ce que gère le logiciel, à utiliser avec précaution. En particulier on peut :
  • Créer/modifier les catégories.
  • Créer/modifier les unités.
  • Supprimer un foyer/fournisseur/référence/catégorie. Une fois un objet supprimé on ne peut pas le retrouver, à utiliser avec modération donc. Attention si l'on supprime un fournisseur / une catégorie cela supprimera aussi tous les produits correspondants (le logiciel demande confirmation avant).
  • Régler le seuil en dessous duquel on ne peut plus faire d'achats (dans "Réglages divers").
  • Changer le texte de la page d'accueil (dans "Réglages divers").
